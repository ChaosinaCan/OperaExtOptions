<static> OptionsPage.addInput(element, resetbutton)
Sets up a new <input> or <select> element. Use this if an element is dynamically added after the options page loads.
Parameters:
{HTMLElement} element
The <input> or <select> element to add
{HTMLElement} resetbutton Optional
A <button> or other clickable element that should reset the setting when clicked. If this parameter is unused and a reset button for the setting already exists on the page, it will be used.
<static> OptionsPage.debugStorage(sortfunction)
Returns a <dl> element containing each of the key-value pairs stored in the web storage object used by the options page's SettingStorage object.
Parameters:
{Function} sortfunction Optional
A function used to sort the list. If unused, the list will be sorted alphabetically.
